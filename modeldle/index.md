# 🤖 Modeldle

A daily Wordle-like guessing game for machine learning models!

## How to Play

1. **Guess a Model** - Search for and select a machine learning model from the list
2. **Check Attributes** - When you guess, you'll see 4 key attributes:
   - Learning Type (Supervised, Unsupervised, etc.)
   - Tasks (regression, classification, clustering, etc.)
   - Interpretability (High, Medium, Low)
   - Date Introduced (Year the model was published)
3. **Match Indicators** - Green checkmarks (✓) show matching attributes, red X's (✗) show mismatches
4. **Win** - Identify the daily model!

**New model every day** - The daily target is selected using the date as a seed, so everyone plays the same model each day.
